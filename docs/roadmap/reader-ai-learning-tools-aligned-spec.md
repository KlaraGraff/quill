# 阅读器 AI 学习工具对齐规格

> 状态：已授权实施；对应功能已进入 `v1.4.0` 发布验证
>
> 日期：2026-07-13
>
> 适用范围：单词查词、短语/句子选区工具、学习卡片、正文标记、笔记、英语等级、AI 服务配置

## 1. 文档用途与决策标记

本文档记录产品讨论中已经对齐的要求，也是 `v1.4.0` 的验收基线。实际实现、迁移和验证结果记录在对应实施计划中。

- **已确认**：用户已经明确提出或确认的产品行为。
- **推荐默认**：当前推荐采用的默认值，实施前仍可调整。
- **待确认**：需要在视觉设计或编码前继续确定的细节。

本轮已获授权修改应用代码、数据库和发布流程；本文仍保留产品决策表述，避免把实现细节混入规格。

## 2. 总体交互模型

### 2.1 文本类型与入口

| 文本类型 | 触发方式 | 首要动作 | 结果容器 |
|---|---|---|---|
| 单个单词 | 单击正文单词 | 立即查词 | 单词学习卡片 |
| 短语/习语 | 选中后右键 | 释义 | 短语学习卡片 |
| 句子/段落 | 选中后右键 | 解读 | 句子学习卡片 |

**已确认：**单词单击后直接查询，不先显示菜单。短语、句子和段落必须先选中，再通过右键菜单选择操作。

### 2.2 选区菜单

短语/习语的推荐默认菜单：

1. 释义
2. 问 AI
3. 收藏
4. 高亮/取消高亮
5. 复制

句子/段落的推荐默认菜单：

1. 解读
2. 问 AI
3. 收藏
4. 高亮/取消高亮
5. 复制

菜单锚定在选区右上方；空间不足时自动翻转到选区下方。菜单项目在“设置 -> 工具 -> 选区菜单”中允许启用、关闭和拖动排序。

### 2.3 释义与翻译去重

**已确认：**不把“释义”和“翻译”作为两个默认并列入口。

- “释义”解释单词或短语在当前语境中的含义和用法。
- “解读”解释句子或段落的核心意思、语法难点和表达作用。
- 翻译合并为学习卡片中的“目标语言译文”模块。
- 对只需要直译的用户，在选区菜单设置中提供默认关闭的“仅翻译”操作。
- 当解释语言与目标语言相同时，目标语言译文默认自动隐藏，避免重复内容。

## 3. 学习卡片结构

### 3.1 统一容器

单词、短语和句子使用同一套卡片视觉和交互结构，不再分别维护重复的弹窗体系。内容模块随文本类型变化。

卡片底部操作统一为：

1. 收藏
2. 问 AI
3. 笔记
4. 复制

**已确认：**不在卡片底部提供“高亮原文”或含义不清楚的“查看上下文”。在查询历史、收藏或笔记页面中，需要跳回阅读位置时统一使用“定位原文”。

### 3.2 单词卡片的必要模块

以下两个模块不能关闭，但可以调整顺序、默认展开状态和内容密度：

1. 单词信息：单词、音标、词性。
2. 当前语境含义：该词在当前句子中的实际意思。

推荐默认顺序为“当前语境含义 -> 单词信息”，落实“先看到这个词在当前句子里是什么意思”的阅读目标。两个必要模块仍可由用户自行调换；其他模块均可关闭，因此最简卡片可以只显示这两个模块。

### 3.3 单词卡片的可选模块

| 模块 | 推荐默认 | 用途 |
|---|---:|---|
| 目标语言译文 | 按语言自动决定 | 在解释语言之外提供简短目标语言翻译 |
| 常见释义 | 开启 | 按常见程度排列其他常用含义 |
| 常用搭配 | 开启 | 高频搭配、介词组合和固定表达 |
| 词形与构词 | 开启 | 原形、变形、派生词、词根词缀 |
| 句中语法作用 | 开启 | 说明该词在当前句子中的语法成分和关系 |
| 近义词辨析 | 关闭 | 解释容易混淆的近义表达 |
| 语气与使用场景 | 关闭 | 标注口语、正式、文学、贬义、过时等语域 |
| 记忆辅助 | 关闭 | 提供可靠、简短的构词、拼写或易混词助记 |
| 原文片段 | 关闭 | 在脱离阅读页查看记录时补充原文 |

每个常见释义可以带双语例句。例句数量在工具设置中配置为 0-3 条，推荐默认 1 条。

“记忆辅助”不是定时复习提醒。它只提供简短助记，例如构词关系、可靠的拼写提示或易混词对比；不得编造词源或生成牵强故事。定时复习属于独立的 SRS 功能。

### 3.4 句子/段落卡片模块

| 模块 | 推荐默认 | 用途 |
|---|---:|---|
| 当前语境理解 | 必须显示 | 先说明这句话在当前段落中的核心意思 |
| 目标语言译文 | 按语言自动决定 | 提供目标语言译文，不重复同语言解释 |
| 重难点语法 | 开启 | 分析主干、从句、修饰、倒装、省略等 |
| 重点词语 | 开启 | 讲解超出用户等级且影响理解的词语 |
| 固定表达与习语 | 开启 | 解释不能按字面理解的表达 |
| 指代关系 | 开启 | 说明 it、this、they、which 等具体指代 |
| 可复用句型 | 关闭 | 提炼适合写作或口语模仿的句型 |
| 语气与写作效果 | 关闭 | 分析强调、暗示、讽刺和文学效果 |
| 原文片段 | 关闭 | 仅在历史记录等脱离正文的场景补充原文 |

重点词语先按“对当前句子理解的重要程度”排序，同等重要时再按日常常见程度排序，避免简单高频词挤占真正难点的位置。

工具设置同时提供：

- 每次重点词语数量：1-8，推荐默认 3。
- 每个重点词的双语例句数量：0-3，推荐默认 1。
- 学习者等级：引用“个人资料与语言”中的 CEFR 等级。

## 4. 内容密度

### 4.1 设置位置

内容密度放在“设置 -> 工具 -> 卡片设计”中，并分为：

- 单词卡片
- 短语卡片
- 句子卡片

每类卡片顶部提供“默认内容密度”：精简、标准、详细。每个模块展开配置后还可以选择“跟随默认、精简、标准、详细”。

每个可配置模块拥有以下属性：

1. 是否显示；必要模块的开关锁定为开启。
2. 展示顺序。
3. 默认展开或收起。
4. 内容密度。

设置模块只有在收起状态下才显示拖动手柄，避免编辑内容时误拖。

### 4.2 密度影响范围

内容密度影响 AI 请求需要返回的信息量和卡片的渲染内容，但不影响：

- 模块是否启用。
- 模块展示顺序。
- 卡片底部操作。
- 选区菜单项目。
- 用户自己写的笔记内容。

详细内容必须留在所属模块内部，不得统一堆到卡片顶部。模块在卡片中的位置完全服从用户设置的顺序。

### 4.3 密度说明入口

完整密度矩阵内容较长，不在设置页常态展开。

**推荐默认：**在“默认内容密度”标题旁放置 `CircleHelp` 信息图标：

- 悬停提示“查看内容密度说明”。
- 点击后打开较宽的说明对话框。
- 对话框使用“单词 / 短语 / 句子”标签切换对应矩阵。
- 设置页本身只常态显示当前选项的一句摘要。
- 对话框纯说明，不在其中修改设置，避免两处设置状态不一致。

### 4.4 单词与句子模块密度矩阵

| 模块 | 精简 | 标准 | 详细 |
|---|---|---|---|
| 单词信息 | 单词、一个音标、主要词性 | 增加英美音标、原形 | 增加不同词性、常见词形、语域标签 |
| 当前语境含义 | 一句最直接的含义 | 增加为什么在这里表达这个意思 | 增加语气、隐含含义及与常见义的区别 |
| 常见释义 | 释义列表 | 释义、词性、配置数量的例句 | 增加使用场景、语域和常见搭配 |
| 词形与构词 | 原形和当前词形 | 增加常见派生词 | 增加词根词缀及词形关系 |
| 句中语法 | 标注句中成分 | 解释与其他成分的关系 | 展开从句、修饰范围、省略或倒装 |
| 重点词语 | 词语和简义 | 增加语境义和配置数量的例句 | 增加搭配、易错点和近义区别 |
| 固定表达 | 列出表达和简义 | 增加当前语境用法 | 增加字面义差异、语域和替代表达 |
| 语气与效果 | 给出核心标签 | 用一句话解释效果 | 结合上下文解释作者意图和细微差别 |

示例：如果“词形与构词”排在第五位并设置为详细，词根词缀和词形关系只在第五个模块内出现；不会附加到“单词信息”或“当前语境含义”中。

### 4.5 响应式卡片尺寸

**已确认：**可选模块增加后，卡片必须按文本类型和内容密度自动变宽，不能继续使用过窄、无限向下增长的固定弹窗。

| 卡片类型 | 精简 | 标准 | 详细 |
|---|---:|---:|---:|
| 单词 | 380px | 480px | 600px |
| 短语 | 420px | 520px | 620px |
| 句子/段落 | 460px | 560px | 680px |

以上是目标宽度，实际宽度不得超过当前阅读区域宽度减去 24px。卡片打开时确定宽度，流式输出期间保持稳定，避免左右跳动。

- 最大高度为窗口高度的 75%，超出后只有模块内容区滚动。
- 标题区和底部操作区保持可见。
- 窄窗口下转为左右各留 12px 的近全宽面板。
- 自动避开点击词或选区；上下均无足够空间时显示在阅读区中央。
- 模块以全宽分区和分隔线组织，不在卡片内嵌套装饰卡片。
- 工具设置提供“卡片宽度：自动 / 紧凑 / 宽”，推荐默认“自动”。

## 5. 动态预览

“设置 -> 工具”提供单词、短语、句子三个预览标签。预览必须即时响应：

- 模块开关和顺序。
- 默认展开/收起状态。
- 精简、标准、详细密度。
- 双语例句数量和重点词数量。
- 解释语言、目标语言和 CEFR 等级。
- 选区菜单项目及顺序。

即时预览使用本地固定的结构化样例，不因每次拖动或开关而消耗 API。另设“生成真实预览”按钮，由用户主动调用当前 AI 服务验证真实输出。

## 6. 高亮与查词标记

### 6.1 手动选区高亮

**已确认：**不提供“调整高亮”这一额外状态。

- 如果选区尚未被完整高亮，菜单显示“高亮”。执行后覆盖整个选区，并合并相邻或重叠的同色高亮。
- 如果选区已经完整高亮，菜单显示“取消高亮”。
- 如果原先只有一部分高亮，第一次操作仍是把整个选区高亮；再次选择相同范围时即可取消。
- 从一段更大的高亮中取消较小选区时，只移除所选范围，保留范围外内容。

### 6.2 查词后的全书单词标记

**已确认：**单击查词的单词默认自动高亮，并在本书后续出现时继续高亮。用户可以在工具设置中关闭这一行为。

推荐匹配行为：

- 忽略大小写。
- 只匹配完整单词，不匹配其他单词内部的相同字符。
- 默认只匹配相同拼写。
- 可选“同一词族”模式，用于 run、runs、ran、running 等词形。

这类标记是“全书单词规则”，不是为每次出现创建一条手动高亮记录。取消时显示明确操作“取消全书‘word’高亮”，并移除该词全部自动标记及后续标记规则。

全书单词标记与手动高亮是两个独立图层。取消单词规则不能删除包含该词的整句手动高亮。

工具设置提供：

- 查词后自动高亮，推荐默认开启。
- 相同拼写/同一词族匹配模式。
- 自动标记颜色和透明度。
- 清除本书全部查词标记。

## 7. 收藏与笔记

### 7.1 概念边界

- 收藏：把内容加入后续查看或学习列表。
- 笔记：保存用户自己的理解、疑问或补充。
- 查询历史：记录发生过的 AI 查询。
- 高亮：在正文中建立视觉标记。

四者不得因为一次操作而隐式互相创建，除非未来提供明确且默认关闭的自动化选项。

### 7.2 笔记入口与卡片回显

点击卡片底部“笔记”后，在卡片内展开简洁编辑区。

- 没有笔记时显示新增编辑区。
- 已有笔记时，在底部操作区上方自动出现“我的笔记”模块。
- 卡片默认显示最近两条，并提供“查看全部”。
- 支持新增、编辑、删除和修改时间显示。

推荐归属规则：

- 单词笔记默认绑定“本书中的标准化单词”，以后在本书任意位置查询同词时回显。
- 可选把单词笔记设为“所有书籍通用”。
- 短语和句子笔记默认绑定原文选区、书籍和位置。
- 短语/句子再次从相同位置打开时回显原笔记。

### 7.3 统一笔记页面

在侧边栏“已保存”下新增“笔记”页面，统一查看所有笔记，并支持：

- 全文搜索。
- 按书籍、单词、短语、句子、标签和时间筛选。
- 编辑、删除和批量导出。
- 点击“定位原文”返回对应阅读位置。
- 通过现有同步体系跨设备同步；必须先补齐事件、合并、快照和迁移，不能只同步一半数据。

删除书籍时应询问用户：同时删除关联笔记，或保留为失去原文链接的独立学习资料。

## 8. 英语等级与解释语言

### 8.1 CEFR 等级含义

| 等级 | 面向用户的说明 | 模型英文表达约束 |
|---|---|---|
| A1 | 能理解最基础单词和极短句 | 极短句和基础词汇，一次解释一个核心意思 |
| A2 | 能理解常见生活表达和简单段落 | 常见词汇、简单连接词，避免抽象术语 |
| B1 | 能独立阅读清晰的日常英语 | 自然但直接的英语，难词必须就地说明 |
| B2 | 能理解抽象话题和一般文学表达 | 可解释语气和抽象含义，控制复杂句长度 |
| C1 | 能理解复杂文章及细微差别 | 允许精确术语和复杂句式，保持解释清晰 |
| C2 | 接近熟练母语阅读能力 | 可分析隐喻、文体和高度抽象内容 |

等级设置放在“设置 -> 个人资料与语言”，供查词、句子解读、重点词筛选和例句生成共同使用。

### 8.2 低等级不强制纯英文

**已确认：采用“自适应双语”，不直接禁用英文解释。**

原因：A1/A2 阶段强行只用简单英文可能造成释义失真，但完全禁止英文又会失去可理解输入的学习机会。

解释模式提供：

1. 中文解释。
2. 英文解释。
3. 自适应双语，推荐默认。

自适应双语的推荐行为：

| 等级 | 默认输出策略 |
|---|---|
| A1-A2 | 中文准确释义为主，附极简英文释义和英文例句 |
| B1 | 简单英文为主；抽象、易误解之处补充简短中文 |
| B2 | 英文为主；中文仅用于细微差别或用户启用的目标译文 |
| C1-C2 | 英文精确解释为主；中文译文按模块设置显示 |

用户仍可强制选择纯英文。A1/A2 首次选择纯英文时给出一次非阻断提示：“当前等级使用纯英文可能降低解释准确性，建议使用自适应双语。”用户确认后不再反复提示。

模型提示必须要求：英文部分使用用户等级可理解的词汇；不可避免的高级词要立即用更简单的英语或中文说明。双语不是逐句机械重复，而是用中文保证准确、用英文提供可理解输入。

### 8.3 根据考试成绩估算

等级设置提供“手动选择”和“根据考试成绩估算”两种模式。

成绩录入使用“添加考试成绩”，而不是一次展示大量输入框：

- IELTS
- TOEFL iBT
- TOEIC Listening & Reading
- Cambridge English Scale
- Duolingo English Test
- CET-4/CET-6，标记为非官方近似估算

每条记录包含考试类型、总分、可选阅读分数和考试日期。估算原则：

- 有阅读单项时，以阅读分数为主要依据。
- 多项成绩同时存在时综合官方映射可信度、考试日期和阅读分数。
- 结果相差两个 CEFR 等级以上时，不静默决定，显示估算范围并让用户确认。
- 始终允许手动覆盖，页面显示估算依据。
- CET-4/CET-6 没有权威的一对一 CEFR 换算，不能伪装成精确结果。

精确分数映射表在实施前根据各考试机构的当前官方对照资料单独固化，并记录资料版本和发布日期。

## 9. 朗读

朗读不作为本轮核心功能的发布阻塞项。待查词、卡片、笔记、高亮和 AI 服务稳定后，再使用 macOS/Windows 系统语音实现：

- 不消耗文字模型 API。
- 不要求新的 API Key。
- 扬声器图标放在单词信息附近，不占用统一的卡片底部操作。
- 工具设置提供系统声音和语速；系统无可用语音时隐藏入口。

当前界面不得预留不可用按钮。云端高质量 TTS 作为后续独立能力评估，不与当前文字 AI 服务配置混用。

## 10. AI 服务配置

### 10.1 完整服务档案

顶层对象命名为“AI 服务”，每套服务完整包含：

- 名称。
- Provider/协议。
- Base URL。
- 一个主 API Key，可选多个备用 Key。
- Model。
- 温度等必要生成参数。
- 启用状态。
- 最近测试状态和延迟。

借鉴 CC-Switch 的完整 Provider 折叠卡片，但不引入其 CLI 配置写入、代理、系统托盘、用量统计等与阅读器无关的功能。

### 10.2 折叠卡片与使用顺序

折叠状态推荐布局：

```text
拖动手柄 | 顺序 | 服务名称 | Provider · Model | 状态 | 延迟 | 测试图标 | 展开
```

**已确认：**服务顺序直接通过折叠卡片拖动调整，不在展开内容中设置数字“优先级”。

- 第一项是主服务。
- 第二项是第一个备用服务。
- 后续依次回退。
- 只在卡片收起时允许拖动。
- 用户可单独停用某套服务而不删除配置。
- 界面文案使用“使用顺序”，不使用难以理解的“优先级”。

### 10.3 故障切换

推荐路由顺序：

1. 先尝试当前服务的主 Key。
2. 在尚未输出任何内容且错误允许重试时，尝试该服务的备用 Key。
3. 当前服务不可用时，切换到下一套完整 AI 服务（包括其 URL、Key 和 Model）。
4. 一旦已经向用户输出内容，不中途切换服务，避免拼接不同模型的答案。

鉴权失败、限流、额度、网络和 5xx 分别记录健康状态及冷却时间。请求参数错误不应无限轮换。

### 10.4 获取模型列表

模型输入框旁放置带提示的模型列表/刷新图标：

- 使用当前表单中已经填写但尚未保存的 URL 和最高顺序可用 Key。
- OpenAI-compatible 服务优先尝试标准模型列表端点。
- 返回结果使用可搜索下拉菜单选择。
- 始终保留手动输入，因为部分兼容服务不提供模型列表接口。
- 获取模型失败不等于推理接口不可用。
- Key 只由 Rust 后端读取和使用，不返回 WebView 或写入日志。

### 10.5 连接测试

连接测试放在折叠服务卡片中，使用无文字的 `Activity`/脉冲图标：

- 悬停提示“测试连接”，并提供无障碍名称。
- 点击后向当前 Model 发送最小真实推理请求，而不只请求模型列表。
- 测试中显示旋转状态。
- 成功后显示绿色状态点、首次响应时间和总耗时。
- 失败后显示红色状态点；展开卡片查看脱敏后的具体原因。

测试不得泄露 Key，也不能把模型列表请求成功误报为完整查词链路成功。

## 11. 已锁定的实施默认值

以下实施默认值已经确认：

1. 视觉沿用现有 Quill 主题 Token；自动查词标记与手动高亮保持可辨识的独立样式，具体颜色通过主题和视觉 QA 校准。
2. 首版全书单词标记只匹配相同拼写并忽略大小写；“同一词族”保留设置入口或数据扩展点，不在首版实现不可靠的自制词形还原。
3. 笔记首版使用支持段落换行的纯文本；数据保留 `content_format` 字段，为以后升级留出兼容空间。
4. 单词笔记默认作用于“本书中的标准化单词”；编辑区提供“本书 / 所有书籍”范围选择。短语和句子笔记绑定具体选区与位置。
5. 考试换算优先使用当前官方 CEFR 对照资料并记录资料版本；CET-4/CET-6 必须标注近似估算，最终结果允许手动覆盖。
6. 解释模式默认“自适应双语”。A1/A2 以中文准确释义为主，B1 以简单英文为主并按需补中文，B2 以上以英文为主。
7. 系统朗读延后，不阻塞本轮核心功能，也不显示尚未可用的入口。
8. 卡片采用 4.5 节的自动响应式宽度、内部滚动和稳定尺寸约束。
